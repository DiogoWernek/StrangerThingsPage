# Stranger Things Landing Page

A front-end project developed during the DevArt event, focused on enhancing animation skills, visual design, and user experience.
This landing page is inspired by Stranger Things and aims to deliver a polished, immersive, and visually engaging interface.

# Overview

The main objective of this project was to apply modern animation techniques using GSAP, while strengthening design principles such as visual hierarchy, motion consistency, and storytelling through interaction.

The project demonstrates the ability to combine clean front-end code with well-structured animations to create a high-quality web experience.
